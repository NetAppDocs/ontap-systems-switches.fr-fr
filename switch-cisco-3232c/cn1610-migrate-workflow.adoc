= Flux de travail pour la migration des commutateurs CN1610 vers les commutateurs Nexus 3232C
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Suivez ces étapes de flux de travail pour migrer vos commutateurs CN1610 vers des commutateurs Cisco Nexus 3232C.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Un"]link:cn1610-migrate-to-3232c-overview.html["Exigences de migration"]
[role="quick-margin-para"]
Consultez les informations relatives aux commutateurs d'exemple pour le processus de migration.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Deux"]link:cn1610-prepare-to-migrate.html["Préparez-vous à la migration"]
[role="quick-margin-para"]
Préparez vos commutateurs CN1610 pour la migration vers les commutateurs Nexus 3232C.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Trois"]link:cn1610-replace-CL2.html["Remplacer le commutateur de groupe CL2"]
[role="quick-margin-para"]
Remplacez le commutateur de cluster CL2 par le nouveau commutateur Nexus 3232C C2.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatre"]link:cn1610-replace-CL1.html["Remplacer le commutateur de groupe CL1"]
[role="quick-margin-para"]
Remplacez le commutateur de cluster CL1 par le nouveau commutateur Nexus 3232C C1.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinq"]link:cn1610-complete-migration.html["Finalisez votre migration"]
[role="quick-margin-para"]
Finalisez votre migration vers les nouveaux commutateurs Nexus 3232C.
