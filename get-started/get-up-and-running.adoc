= Soyez opérationnel avec les systèmes AFX, les clusters, le stockage et les commutateurs partagés
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Pour démarrer avec les systèmes AFX, les clusters, le stockage et les commutateurs partagés, vous installez des composants matériels et configurez votre commutateur.

Le déploiement du commutateur implique le flux de travail suivant.

[role="quick-margin-para"]
