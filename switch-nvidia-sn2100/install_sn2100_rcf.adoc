= Installez le script RCF
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Avant d'installer le script RCF, assurez-vous que les éléments suivants sont disponibles sur le commutateur :

* Cumulus Linux 4.4.2 est installé.
* Adresse IP, masque de sous-réseau et passerelle par défaut définis via DHCP ou configurés manuellement.



IMPORTANT: Voir link:install_cabling_config_considerations_sn2100.html["Considérations relatives au câblage et à la configuration"^] pour mises en garde et autres détails.

Deux scripts RCF sont disponibles pour les applications de stockage et de mise en cluster. La procédure pour chaque est la même.

* Mise en cluster : *MSN2100-RCF-v1.8-Cluster*
* Stockage : *MSN2100-RCF-v1.8-Storage*



NOTE: L'exemple de procédure suivant montre comment télécharger et appliquer le script RCF pour les commutateurs de cluster.


NOTE: Exemple de sortie de commande utilise l'adresse IP de gestion de commutateur 10.233.204.71, le masque de réseau 255.255.254.0 et la passerelle par défaut 10.233.204.1.

+
Le script RCF effectue les étapes suivantes :

+
.. Met à jour la bannière MOTD
.. Désactive apt-get pour les mises à jour du système d'exploitation
.. Définit les interfaces écorchées et non écorchées
.. Configure les interfaces et le SNMP
.. Désactive CDP
.. Modifie la configuration LLDP
.. Ajoute une configuration RoCE
.. Modifie la configuration RoCE pour la haute disponibilité et le RDMA des clusters
.. Redémarre le commutateur
+

NOTE: Pour tout problème de script FCR python qui ne peut pas être corrigé, contactez https://mysupport.netapp.com/["Support NetApp"^] pour obtenir de l'aide.
