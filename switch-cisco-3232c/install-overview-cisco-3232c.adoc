= Flux de travail d'installation et de configuration pour les commutateurs Cisco Nexus 3232C
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Les commutateurs Cisco Nexus 3232C peuvent être utilisés comme commutateurs de cluster dans votre cluster AFF ou FAS . Les commutateurs de cluster vous permettent de créer des clusters ONTAP avec plus de deux nœuds.

Suivez ces étapes de flux de travail pour installer et configurer votre commutateur Cisco Nexus 3232C.
