= Flux de travail d'installation du logiciel pour les commutateurs de stockage Cisco Nexus 9336C-FX2 et 9336C-FX2-T
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Pour installer et configurer le logiciel des commutateurs de stockage Cisco Nexus 9336C-FX2 et 9336C-FX2-T, procédez comme suit :

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Un"]link:setup-switch-9336c-storage.html["Configurez le commutateur"]
[role="quick-margin-para"]
Configurez les commutateurs de stockage 9336C-FX2 et 9336C-FX2-T.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Deux"]link:install-nxos-overview-9336c-storage.html["Préparez-vous à installer le logiciel NX-OS et RCF."]
[role="quick-margin-para"]
Le logiciel Cisco NX-OS et les fichiers de configuration de référence (RCF) doivent être installés sur les commutateurs de stockage Cisco 9336C-FX2 et 9336C-FX2-T.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Trois"]link:install-nxos-software-9336c-storage.html["Installez ou mettez à niveau le logiciel NX-OS"]
[role="quick-margin-para"]
Téléchargez et installez ou mettez à niveau le logiciel NX-OS sur les commutateurs de stockage Cisco 9336C-FX2 et 9336C-FX2-T.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatre"]link:install-upgrade-rcf-overview-9336c-storage.html["Installer ou mettre à niveau le RCF"]
[role="quick-margin-para"]
Installez ou mettez à niveau le RCF après avoir configuré les commutateurs Cisco 9336C-FX2 et 9336C-FX2-T pour la première fois. Vous pouvez également utiliser cette procédure pour mettre à niveau votre version RCF.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinq"]link:configure-ssh-keys.html["Vérifier la configuration SSH"]
[role="quick-margin-para"]
Vérifiez que SSH est activé sur les commutateurs pour utiliser les fonctionnalités de surveillance de l’état du commutateur Ethernet (CSHM) et de collecte de journaux.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["Six"]link:reset-switch-9336c-storage.html["Réinitialiser le commutateur aux paramètres d'usine"]
[role="quick-margin-para"]
Effacez les paramètres des commutateurs de stockage 9336C-FX2 et 9336C-FX2-T.
