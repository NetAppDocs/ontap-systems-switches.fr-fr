= Flux de travail d'installation et de configuration pour les commutateurs de stockage Cisco Nexus 9336C-FX2 9336C-FX2-T
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Les commutateurs Cisco Nexus 9336C-FX2 et 9336C-FX2-T font partie de la plate-forme Cisco Nexus 9000 et peuvent être installés dans une armoire système NetApp .

Cisco Nexus 9336C-FX2 (36 ports) est un commutateur de cluster/stockage/données à haute densité de ports. Cisco Nexus 9336C-FX2-T (12 ports) est un commutateur hautes performances à faible densité de ports qui prend en charge les configurations 10/25/40/100GbE.
