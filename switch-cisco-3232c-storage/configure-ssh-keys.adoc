= Vérifiez votre configuration SSH
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si vous utilisez les fonctionnalités Ethernet Switch Health Monitor (CSHM) et de collecte de journaux, vérifiez que SSH et les clés SSH sont activés sur les commutateurs.

.Étapes
. Vérifiez que SSH est activé :
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. Vérifiez que les clés SSH sont activées :
+
`show ssh key`

+

IMPORTANT: Lors de l'activation du FIPS, vous devez modifier le nombre de bits à 256 sur le commutateur à l'aide de la commande `ssh key ecdsa 256 force` .  Voir https://docs.netapp.com/us-en/ontap/networking/configure_network_security_using_federal_information_processing_standards_@fips@.html#enable-fips["Configurer la sécurité du réseau à l'aide de FIPS"^] pour plus de détails.
