= Flux de travail d'installation matérielle pour les commutateurs de stockage Cisco Nexus 9336C-FX2 et 9336C-FX2-T
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Pour installer et configurer le matériel des commutateurs de stockage 9336C-FX2 et 9336C-FX2-T, procédez comme suit :

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Un"]link:setup-worksheet-9336c-storage.html["Complétez la fiche de câblage"]
[role="quick-margin-para"]
La fiche de câblage type fournit des exemples d'affectations de ports recommandées entre les commutateurs et les contrôleurs.  La feuille de calcul vierge fournit un modèle que vous pouvez utiliser pour configurer votre cluster.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Deux"]link:install-9336c-storage.html["Installez l'interrupteur"]
[role="quick-margin-para"]
Installez les commutateurs de stockage 9336C-FX2 et 9336C-FX2-T.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Trois"]link:install-switch-and-passthrough-panel-9336c-storage.html["Installez le commutateur dans une armoire NetApp."]
[role="quick-margin-para"]
Installez les commutateurs 9336C-FX2 et 9336C-FX2-T et le panneau de passage dans une armoire NetApp selon les besoins.
