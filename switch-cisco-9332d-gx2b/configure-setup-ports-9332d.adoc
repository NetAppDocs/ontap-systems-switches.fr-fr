= Détails du port pour les commutateurs Cisco Nexus 9332D-GX2B
:allow-uri-read: 


[role="lead"]
Le commutateur Cisco Nexus 9332D-GX2B fait partie de la plate-forme Cisco Nexus 9000. Les commutateurs système AFX vous permettent de créer des clusters ONTAP avec plus de deux nœuds.



== Détails du port Cisco Nexus 9332D-GX2B

[cols="1,2"]
|===
| Ports | Description 


 a| 
Ethernet1/1/1-4 vers Ethernet1/30/1-4
 a| 
Mode 4x100GbE pour les connexions cluster, HA et stockage



 a| 
Ethernet1/31, Ethernet1/32
 a| 
ISL 400GbE

|===
Voir le https://hwu.netapp.com["Hardware Universe"^] pour plus de détails sur les ports utilisés par la plateforme. Voir link:https://kb.netapp.com/Support/General_Support/What_additional_information_do_I_need_to_install_my_equipment_that_is_not_in_HWU["De quelles informations supplémentaires ai-je besoin pour installer mon équipement qui ne figure pas dans HWU ?"^] pour plus d'informations sur les exigences d'installation du commutateur.
